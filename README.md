python-projects-beginners
A collection of easy Python projects for beginners. This repository contains simple games that you can run directly from your terminal.
## How to Run (Bash/Terminal)
Follow these steps to run any of the games in a Bash terminal (Git Bash, WSL, macOS, or Linux).
### 1. Clone and Navigate
```bash
git clone https://github.com/afthahmm590-png/python-projects-beginners.git
cd python-projects-beginners
```
### 2. Run the Games
Use the following commands to start each game:
- **Dice Rolling Game**:
  ```bash
  python dice_rolling_game.py
  ```
- **Number Guessing Game**:
  ```bash
  python number_guessing_game.py
  ```
- **Rock Paper Scissors**:
  ```bash
  python rock_paper_scissor.py
  ```
- **To Do List**:
  ```bash
  python to_do_list.py
  ```
> [!TIP]
> If `python` doesn't work, try using `python3` instead.
## Projects Included
1.  **Dice Rolling Game**: A fun way to simulate rolling dice.
2.  **Number Guessing Game**: Test your luck guessing numbers between 1 and 100.
3.  **Rock Paper Scissors**: Play the classic strategy game against the computer.
